# Data Licences

Datasets located within `data/` are included in this package under their respective licences.

## globorisk_coefs, globorisk_cvdr, globorisk_rf

Datasets prefixed `globorisk_` are derived from the `globorisk` R package, which is available from [CRAN](https://cran.r-project.org/) and [GitHub](https://boyercb.github.io/globorisk).

They are provided under globorisk's MIT licence, the full licence text is available within `LICENSE_globorisk.md`.

## pop_snapshot, lifetable_data

Datasets `pop_snapshot` and `lifetable_data` are derived from the United Nations World Population Prospects 2024 package (`wpp2024`), which is available from [GitHub](https://github.com/PPgp/wpp2024).

They are provided under wpp2024's CC-BY licence, the full licence text is available within `LICENSE_wpp2024.md`.

## bmi_fits

The dataset `bmi_fits` is derived from the GitHub repository [`petedodd/bmitb`](https://github.com/petedodd/bmitb).

It is provided under the bmitb's MIT licence, the full licence text is available within `LICENSE_bmitb.md`.
